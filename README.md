# Hospital Emergency Queue Management

A simple web app for managing patients in an emergency queue by priority.

## Features
- Add patients with a name and priority level
- Sort patients so critical cases appear first
- Display the current queue in a clean table

## Project Structure
- index.html: Main page structure
- style.css: Styling for the interface
- script.js: Queue logic and rendering

## How to Run
1. Open index.html in a browser, or
2. Serve the project folder with a simple local server

## Usage
- Enter a patient name
- Choose a priority level: Critical, High, Medium, or Low
- Click Add Patient to update the queue
